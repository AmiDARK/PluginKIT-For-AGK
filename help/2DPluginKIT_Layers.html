PluginKIT Commands Sets : 2DPluginKIT Images Collections Commands

The Images Collections Commands set was created to simplify support of Images Collections for Tiles system of 2DPluginKIT.
NewLayer (As PKLayer_Type) = PKCreateNewLayer( LayerName (As String), LayerWidth (As Integer), LayerHeight (As Integer) )
This method will create a new tile based layer.
LayerWidth, LayerHeight are the width and height of the layer in amount of tiles.
To know the true pixel sizes of the layer you should multiply LayerWidht and LayerHeight respectively by TilesWidth and TilesHeight.
PKDeleteLayer( LayerID (As Integer) )
This method will delete the layer LayerID. All the layer with ID higher than this layer will see their ID be decreased by 1. To avoid problems of coherencies when deleting some layers (and not all at the same time), I have added a new method to use layer by Name instead of by ID.
NewLayer (As PKLayer_Type) = PKCloneLayer( SourceLayerID (As Integer) )
This method will create a new layer as a copy/clone of the chosen layer SourceLayerID.
isExists (As Boolean) = PKGetLayerExists( LayerID (As Integer) )
Thie method will return 'TRUE' if the specified layer LayerID exists. Otherwise 'False'.
PKSetLayerTile( LayerID (As Integer), X (As Integer), Y (As Integer), TileID (As Integer) )
This method will update the tile at coordinates X,Y of the Layer LayerID, to use the tile TileID.
PKSetLayerArea( LayerID (As Integer), aLeft (As Integer), aTop (As Integer), aRight (As Integer), aBottom (As Integer) )
This method defines the area in the screen, where the layer LayerID can be displayed. To achieve this, it define a box with coordinates defined by 2 dots : aLeft,aTop and aRight,aBottom.
PKSetLayerViewPosition( LayerID (As Integer), XPos (As Float), YPos (As Float) )
PKSetLayerViewPositionX( LayerID As Integer, XPos As Float )
PKSetLayerViewPositionY( LayerID As Integer, YPos As Float )
This method will define the scrolling value of the layer LayerID.
XPos,YPos are the coordinates, in pixels inside the Layer, to define the top,left in layer coordinate to start drawing the layer on screen.
You can set XPos and YPos at the same time, or update only XPos or YPos.
PKSetLayerUVMode( LayerID (As Intege), XC (As Boolean), YC (As Boolean) )
This method will allow user to set if Layer LayerID will loop in X axis (XC) and on Y axis (YC) ). 'TRUE' will enable looping, 'FALSE' will disable it.
PKSetLayerTilesSizes( LayerID (As Integer), Width (As Integer), Height (As Integer) )
This method is used to define the sizes (width and height) in pixels of each tile of the layer LayerID.
PKScrollLayer( LayerID (As Integer), XScroll (As Integer), YScroll (As Integer) )
PKScrollLayerX( LayerID (As Integer), XScroll (As Integer) )
PKScrollLayerY( LayerID (As Integer), YScroll (As Integer) )
This method will scroll the layer LayerID of XScroll,YScroll pixels from its current scrolling coordinates.
You can update both XScroll and YScrool values, or separately XScroll or YScroll.
PKTraceAllLayers()
This method will display all the existing layers on screen using all their parameters and contents.
PKSetLayerVisible( LayerID (As Integer), VisibleMODE (As Boolean) )
This method will enable (VisibleMODE=TRUE) the drawing of layer LayerID, and disable it (VisibleMODE=FALSE)
isVisible (As Boolean) = PKisLayerVisible( LayerID (As Integer) )
This method will return 'TRUE' if layer is enabled, otherwise 'FALSE'.
LayerXTilesWidth (As Integer) = PKGetLayerWidth( LayerID As Integer )
This method will return the amount of tiles that the layer LayerID contains on each of its tiles lines.
LayerXTilesHeight (As Integer) = PKGetLayerHeight( LayerID As Integer )
This method will return the amount of tiles that the layer LayerID contains on each of its tiles columns.
TileWidth (As Integer) = PKGetLayerTileWidth( LayerID (As Integer) )
This method will return the width in pixels of a layer LayerID tile.
TileHeight (As Integer) = PKGetLayerTileHeight( LayerID (As Integer) )
This method will return the height in pixels of a layer LayerID tile.
IsCycling (As Boolean) = PKGetLayerXCycling( LayerID (As Integer) )
This method will return 'TRUE' if the cycling mode is enabled on X axis for the layer LayerID. Otherwise 'FALSE'.
IsCycling (As Boolean) = PKGetLayerYCycling( LayerID (As Integer) )
This method will return 'TRUE' if the cycling mode is enabled on Y axis for the layer LayerID. Otherwise 'FALSE'.
XScrollPos (As Integer) = PKGetLayerXScroll( LayerID (As Integer) )
This method will return the current XScroll Position (in pixels) of the layer LayerID.
YScrollPos (As Integer) = PKGetLayerYScroll( LayerID (As Integer) )
This method will return the current YScroll Position (in pixels) of the layer LayerID.
TileID (As Integer) = PKGetLayerTileID( LayerID (As Integer), X (As Integer), Y (As Integer) )
This method will return the ID of the tile set at tile coordinates X,Y inside the layer LayerID. This method will return -1 if no tiles are set in the specified coordinates.
TileID (As Integer) = PKGetLayerTileIDFromCoordinates( LayerID (As Integer), X (As Float), Y (As Float) )
This method will return the ID of the tile set at pixels coordinates X,Y inside the layer LayerID. This method will return -1 if no tiles are set in the specified coordinates. Pixels coordinates are not relative to screen area in which the layer is drawn, but relative to coordinates 0,0 of the layer itself.
PKSetLayerTileIDFromCoordinates( LayerID (As Integer), X (As Float), Y (As Float), TileID (As Integer) )
This method allow to update the TileID of a tile of the layer LayerID. Coordinates X,Y of the Tile to modify are given in pixels relatives to the top,left (0,0) start coordinates of the layer in absolute coordinates (from the true layer definition coordinates)
PKSetLayerScrollMode( LayerID (As Integer), ScrollMODE (As Integer) )
This method allow to define auto-scrolling mode for layer LayerID in relation with layer defined as GameLayer.
0 mean that this layer will keep it's current scroll value (manual scroll) when Game Layer will be scrolled
1 mean that this layer will be scrolled XSpeed,YSpeed times the scroll value of Game Layer
2 mean that this layer will be scrolled the same speed than the Game Layer
3 mean that this layer will always be displayed from it's top,left (0,0) coordinates (no scrolling)
PKSetLayerScrollSpeedXY( LayerID (As Integer), XSCR (As Float), YSCR (As Float) )
PKSetLayerScrollX( LayerID (As Integer), XSCR (As Float) )
PKSetLayerScrollY( LayerID (As Integer), YSCR (As Float) )
This method allow user to define the XSpeed,Yspeed factor used to scroll this layer (and used when the command PKSetLayerScrollMode set ScrollMODE to 1) when GameLayer is scrolled.
PKSetGameLayer( LayerID (As Integer) )
Makes the specified layer to become the GameLayer. it is useful to makes other layers movements being dependant on this layer.
LayerID (As Integer) = PKGetGameLayer()
Return the LayerID of the layer that is currently defined as GameLayer.
If no one is defined, the method return -1.
ScrollMODE (As Integer) = PKGetLayerScrollMode( LayerID (As Integer) )
This method will return the type of Scroll mode defined for the layer LayerID when this layer is not the GameLayer
XSpeed (As Integer) = PKGetLayerXSpeed( LayerID (As Integer) )
This method return the speed on X axis to calculate the scrolling applied to this layer LayerID when the GameLayer is scrolled
YSpeed (As Integer) = PKGetLayerYSpeed( LayerID (As Integer) )
This method return the speed on Y axis to calculate the scrolling applied to this layer LayerID when the GameLayer is scrolled
PKSetGameLayerScroll( XPosF (As Float), YPosF (As Float) )
This methods makes the layer defined to be the GameLayer to be displayed scrolled by XPosF,YPos pixels.
All others layers, depending on ScrollMODE defined will move this way :
0 mean that other layer will keep it's current scroll value (manual scroll) when Game Layer will be scrolled
1 mean that other layer will be scrolled XSpeed,YSpeed times the scroll value of Game Layer
2 mean that other layer will be scrolled the same speed than the Game Layer
3 mean that other layer will always be displayed from it's top,left (0,0) coordinates (no scrolling)
PKWriteLAYERToFILE( ChannelID (As Integer), LayerID (As Integer) )
This method will save the layer LayerID inside a file Channel ChannelID previously opened with OpenToWrite command from AppGameKit
LayerID (As Integer) = PKCreateLAYERFromFile( ChannelID (As Integer) )
This method will create a new layer LayerID from a previously saved layer using the file channel ChannelID previously opened using OpenToRead command from AppGameKit
PKWriteAllLayersToFile( ChannelID (As Integer) )
This method will save all existing layer (visible or not) in their numerical order, inside a file Channel ChannelID previously opened with OpenToWrite command from AppGameKit
LayerCount (As Integer) = PKCreateAllLayersFromFile( ChannelID (As Integer) )
This method will create a new layers from a previously saved layer set using the file channel ChannelID previously opened using OpenToRead command from AppGameKit.
It will return the amount of layers created.
LeftBoundary (As Integer) = PKGetLayerAreaLEFT( LayerID (As Integer) )
This method return the minimal left pixel coordinate of the box inside which the layer will be drawn on screen
RightBoundary (As Integer) = PKGetLayerAreaRIGHT( LayerID (As Integer) )
This method return the maximum right pixel coordinate of the box inside which the layer will be drawn on screen
TopBoundary (As Integer) = PKGetLayerAreaTOP( LayerID (As Integer) )
This method return the minimal upper pixel coordinate of the box inside which the layer will be drawn on screen
BottomBoundary (As Integer) = PKGetLayerAreaBOTTOM( LayerID (As Integer) )
This method return the maximal bottom pixel coordinate of the box inside which the layer will be drawn on screen
PixelColor (As Integer) = PKGetLayerMaskPixel( LayerID (As Integer), XPos (As Integer), YPos (As Integer) )
This method return the color of the mask pixel at the XPos, YPos coordinate inside the layer LayerID.
These coordinates are absolute from the top,left coordinate 0,0 of the Layer.
Mask pixel is a 0-255 value (8 bits) used to define collisions systems between moveable elements, and static tiles.
NormalMapColor (As Integer) = PKGetLayerNMapPixel( LayerID (As Integer), XPos (As Integer), YPos (As Integer) )
This method return the color of the mask pixel at the XPos, YPos coordinate inside the layer LayerID.
These coordinates are absolute from the top,left coordinate 0,0 of the Layer.
Normal mapping pixel is a RGBA 32 bits value used to define lighting systems for tiles and 2D Lights, A normal map image must be defined to obtain a value.
