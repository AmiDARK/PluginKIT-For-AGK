PluginKIT Commands Sets : 2DPluginKIT Animations Commands

The animation Commands set was created to simplify support for animations on Tiles for the Tiles and Layers system of 2DPluginKIT, and for the Sprites (Called Blitter Object) that are linked to Layers.
AnimID (As Integer) = PKNewAnimation( FrameList (As String) )
This method will create a new Animation (returning his ID) from a FrameList.
The Frame list is composed of the header "ANIM" followed by Animation frames Image ID rounded with ';'.
Here is an example of use :
myAnim As Integer
myAnim = PKNewAnimation( "ANIM;4;5;6;7;8;9;" )
This will create a single animation that will start with image 4, and will continue with image 5, 6, 7, 8 and then 9.
The animation is not started when created.
PKSetAnimationFrameShift( AnimationID (As Integer), ImageIDShiftToApply (As Integer) )
This method will makes the Animation 'AnimationID' shift the ids of the Images used for the animation.
If you set FrameList = "ANIM;5;6;7;8;" and used ImageIDShiftToApply = 7, then the animation will start with image 5 + 7 = 12 and then use images : 12,13,14 and then 15.
PKSetAnimationFrame( AnimationID (As Integer), FrameID (As Integer), ImageID (As Integer) )
This method will update the Image 'ImageID' used for the frame 'FrameID' of the Animation 'AnimationID'.
Frame index start at 0. Here is an example of use :
myAnim As Integer
myAnim = PKNewAnimation( "ANIM;4;5;6;7;" )
PKSetAnimationFrame( myAnim, 2, 9 )
Then your animation will play images in this order : 4,5,9,7
ImageID (As Integer) = PKGetAnimationFrameID( AnimationID (As Integer), FrameID (As Integer) )
This method will return the image index number of the frame 'FrameID' of the animation 'AnimationID'.
PKSetAnimationToTile( AnimationID (As Integer), TileID (As Integer) )
This method will affect the chosen animation 'AnimationID' to the tile 'TileID'.
PKSetAnimationToBob( AnimationID (As Integer), BlitterID (As Integer) )
This method will affect the chosen animation 'AnimationID' to the BlitterObject (Bob) 'BlitterID'.
PKSetAnimationToSprite( AnimationID (As Integer), SpriteID (As Integer) )
This method will affect the chosen aimation 'AnimationID' to an AppGameKit Sprite 'SpriteID'
PKSetAnimationSpeed( AnimationID (As Integer), Speed (As Integer) )
This method will update animation speed of a chosen animation 'AnimationID' to the specified value 'Speed' (default=10)
PKPlayAnimation( AnimationID (As Integer) )
This method will tell PluginKIT to start the animation 'AnimationID'
PKStopAnimation( AnimationID (As Integer) )
This method will stop the play of the animation 'AnimationID'
PKLoopAnimation( AnimationID (As Integer), LoopMODE (As Boolean) )
This method will makes the animation 'AnimationID' loop (LoopMODE=TRUE) or stop at ending (LoopMODE=FALSE).
PKDetachAnimationFromObject( AnimationID (As Integer) )
This method will detach the animation 'AnimationID' from any link with Tiles, Bobs or Sprites.
Status (As Boolean) = PKGetAnimationPlaying( AnimationID (As Integer) )
This method will return 'TRUE' if the animation 'AnimationID' is currently playing, otherwise 'FALSE'
isExists (As Boolean) = PKGetAnimationExists( AnimationID (As Integer) )
This method will return 'TRUE' if the animation 'AnimationID' exists, otherwise 'FALSE'.
