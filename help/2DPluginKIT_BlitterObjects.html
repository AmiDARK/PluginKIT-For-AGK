PluginKIT Commands Sets : 2DPluginKIT Blitter Objects (bobs) Commands

The Blitter Objects (bobs) Commands set was created to add support to sprites that can be linked to PluginKIT Layers system.
NewBob (As Integer) = PKLoadBlitterObject( FileName (As String) )
NewBob (As Integer) = PKLoadBlitterObjectEx( FileName (As String), TransparencyFlag (As Integer) )
NewBob (As Integer) = PKLoadBlitterObjectEx2( FileName (As String), TilesXCount (As Integer), TilesYCount (As Integer) )
NewBob (As Integer) = PKLoadBlitterObjectEx3( FileName (As String), TilesXCount (As Integer), TilesYCount (As Integer), TransparencyFlag (As Integer) )
These methods will create new Blitter Objects from an image file.
TransparencyFlag will allow to define the transparency mode used (same than sprites one).
TilesXCount, TilesYCount allow to define the amount of images (frames) available inside the image file used.
NewBob (As Integer) = PKAddBlitterObject( FileName (As String) )
NewBob (As Integer) = PKAddBlitterObjectEx( FileName (As String), TransparencyFlag (As Integer) )
NewBob (As Integer) = PKAddBlitterObjectEx2( FileName (As String), TilesXCount (As Integer), TilesYCount (As Integer) )
NewBob (As Integer) = PKAddBlitterObjectEx3( FileName (As String), TilesXCount (As Integer), TilesYCount (As Integer), TransparencyFlag (As Integer) )
NewBob (As Integer) = PKAddBlitterObjectEx4( BobName (As String), ImageID (As Integer), TilesXCount (As Integer), TilesYCount (As Integer) )
NewBob (As Integer) = PKAddBlitterObjectEx5( BobName (As String), ImageID (As Integer) TilesXCount (As Integer), TilesYCount (As Integer), TransparencyFLAG (As Integer) )
These methods will create new Blitter Objects from an image that was previously loaded.
TransparencyFlag will allow to define the transparency mode used (same than sprites one).
TilesXCount, TilesYCount allow to define the amount of images (frames) available inside the image file used.
NewBob (As Integer) = PKInstanceBlitterObject( SourceBobID (As Integer) )
NewBob (As Integer) = PKInstanceBlitterObjectEx( SourceBobID (As Integer), XPos (As Integer), YPos (As Integer) )
NewBob (As Integer) = PKInstanceBlitterObjectByName( SourceBobName (As String) )
NewBob (As Integer) = PKInstanceBlitterObjectExByName( BobName (As String), XPos (As Integer), YPos (As Integer) )
This method will create a clone bob 'NewBob' from an existing bob 'SourceBobID'. if XPos, YPos are specified, the new bob will be placed at new coordinates instead of keeping the ones from the original bob.
PKAttachBobToLayer( BobID (As Integer), LayerID (As Integer) ) PKAttachBobToLayerBothByName( BobName (As String), LayerID (As Integer) )
This method will attach the bob 'BobID' to an existing layer 'LayerID'.
PKDetachBobFromLayer( BobID (As Integer) ) PKDetachBobByNameFromLayer( BobName (As String) )
This method will detach the bob 'BobID' from any Layer.
PKPositionBOB( BobID (As Integer), XPos (As Integer), YPos (As Integer) ) PKPositionBOBByName( BobName (As String), XPos (As Integer), YPos (As Integer) )
This method will place the bob 'BobID' that the 2D coordinates of the screen XPos, YPos.
PKHideBOB( BobID (As Integer) ) PKHideBOBByName( BobName (As String) )
This method will makes bob 'BobID' to not be drawn on next frames.
PKShowBOB( BobID (As Integer) ) PKShowBOBByName( BobName (As String) )
This method will makes bob 'BobID' being drawn again after a PKHideBOB call.
PKSetBobImage( BobID (As Integer), ImageID (As Integer) ) PKSetBobImageByName( BobName (As String), ImageID (As Integer) )
This method will force the bob 'BobID' to be drawn using image 'ImageID'. It may be better to use PluginKIT Animations system instead of direct image changes excepted if you have complexe animations schemes to use.
PKSetBobFrame( BobID (As Integer), FrameID (As Integer) PKSetBobFrameByName( BobName (As String), (FrameID As Integer) )
If you used TilesXCount,TilesYCount when Loading or Adding the bob, it is then possible to directly uses the frames defined in the Bob Images.
FrameID Start at 0
FrameCount = TilesXCount * TilesYCount.
isExists (As Boolean) = PKGetBobExists( BobID (As Integer) )
This method will return 'TRUE' if the specified bob exists, otherwise 'FALSE'.
XPos (As Integer) = PKGetBobPosX( BobID (As Integer) ) XPos (As Integer) = PKGetBobPosXByName( BobName (As String) )
This method will return the X coordinate of the chosen bob 'BobID'.
YPos (As Integer) = PKGetBobPosY( BobID (As Integer) ) YPos (As Integer) = PKGetBobPosYByName( BobName (As String) )
This method will return the Y coordinate of the chosen bob 'BobID'.
FrameID (As Integer) = PKGetBobFrame( BobID (As Integer) ) FrameID (As Integer) = PKGetBobFrameByName( BobName (As String) )
This method will return the current frame used to display the Bob 'BobID'.
TransparencyFLAG (As Integer) = PKGetBobTransparent( BobID (As Integer) ) TransparencyFLAG (As Integer) = PKGetBobTransparentByName( BobName (As String) )
This method will the return the Transparency mode used to display the sprite. This mode was defined on Loading or Adding blitter object. If not set at creation, it was then set to the default PKSetup transparency mode.
BobID (As Integer) = PKFindBlitterObjectByName( bobName (As String) )
This method will return the BobID of the bob 'bobName' if exists.
